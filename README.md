# Excel Data Analytics Bootcamp Projects

## Overview

This repository contains projects completed as part of the **11-Hour Excel Data Analytics Bootcamp** by Luke Barousse. The course focuses on developing practical data analytics skills using Microsoft Excel, from data cleaning and transformation to dashboard creation and business analysis.

Throughout the bootcamp, I worked with real-world data to build analytical solutions that transform raw datasets into meaningful insights and interactive visualizations.

## Projects

### 1. Data Science Salary Dashboard

An interactive Excel dashboard designed to explore salary trends across data-related careers. The dashboard allows users to analyze compensation by job title, location, and employment type through dynamic visualizations and filtering options.

**Skills Applied**

* Dashboard Design
* Data Visualization
* Pivot Tables & Pivot Charts
* Advanced Excel Functions
* Interactive Reporting

### 2. Data Science Job Market Analysis

A comprehensive analysis of data science job postings aimed at identifying trends in salaries, job demand, and required skills. This project demonstrates the complete analytics workflow, from data preparation to insight generation.

**Skills Applied**

* Data Cleaning
* Power Query
* Power Pivot
* Data Modeling
* Exploratory Data Analysis (EDA)
* Business Insight Generation

## Tools & Technologies
